# 🩸 Smart Blood Report Analyzer
**B.Tech NLP Project** | Python • Streamlit • scikit-learn • pdfplumber

---

## What It Does
Upload a CBC (Complete Blood Count) PDF report → get:
- 🦠 Disease predictions with confidence scores (Random Forest ML)
- 📊 Health score out of 100
- 🥗 Food & diet advice per condition
- 💊 Medication suggestions
- 🏥 Doctor type recommendations
- 📄 Downloadable PDF summary report
- 🇮🇳 Hindi language toggle

## Setup & Run

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Train the ML Model (required once)
```bash
# If ml_model.pkl exists from old version, delete it first:
del ml_model.pkl           # Windows
rm ml_model.pkl            # Linux/Mac

python train_model.py
# Expected output: ~96% accuracy (NOT 99%+ — that was fake)
```

### 3. Run the App
```bash
streamlit run app.py
# Opens at http://localhost:8501
```

## ML Model Details

- **Algorithm**: Random Forest (200 trees)
- **Training samples**: 2,800 (400 × 7 classes)
- **Features**: 14 CBC parameters
- **Accuracy**: ~96% (realistic — not 99%)
- **Classes**: Iron Deficiency Anemia, Leukocytosis, Leukopenia, Normal/Healthy, Polycythemia, Thalassemia Trait, Thrombocytopenia

### Why ~96% and not 99%?
The original dataset had no overlap between classes — IDA Hb max was 10.1 g/dL while Normal Hb min was 13.5 g/dL (a 3.4 g/dL gap). The model just memorized hard thresholds. The fixed dataset introduces mild severity cases that cross reference range boundaries, matching real-world ambiguity.

## Disease Suppression Logic
The ML model's raw output is filtered before display using `DISEASE_KEY_PARAMS`:

```python
# Each disease must have at least one of its key parameters actually abnormal
DISEASE_KEY_PARAMS = {
    "Iron Deficiency Anemia":               {"Hemoglobin", "MCV", "RDW", "MCH"},
    "Thrombocytopenia (Low Platelets)":     {"Platelets"},
    "Polycythemia / Dehydration":           {"PCV", "Hemoglobin", "RBC"},
    "Leukocytosis (High WBC — Infection)":  {"WBC", "Neutrophils"},
    ...
}
```
This prevents the model from showing "Thrombocytopenia" when platelets are 300,000.
